Controle de Processos Industriais: Estação MPS-PA Compacta da Festo

  • CLP
  • Ladder
  • Dashboard
  • Automação
  • Controle
Status: Finalizado (Novembro/2019)

O trabalho tem por objetivo criar um Serviço de Monitoramento e Banco de Dados utilizando-se de conceitos de I4.0, o qual visa fábricas inteligentes. O projeto já está obtendo resultados promissores e revela-se excelente para aplicações de monitoramento e armazenamento em um banco de dados local utilizando-se de aplicações em estruturas de microserviços. A Planta-Piloto está equipada com o sistema de monitoramento para supervisão e armazenamento de dados que poderão ser dispostos para futuras aplicações como Analitycs para aprimoramento de processos e carregar dados em bancos de dados na Nuvem para consultas de qualquer local e com segurança, bastando apenas uma rede de internet disponível.

O trabalho foi bem sucedido onde conseguiu-se realizar a integração do supervisório Labview com a plataforma IoT da Adafruit, fornecendo ao usuário acesso a vários dashboards para monitorar os processos de qualquer local que possua internet, sendo necessário apenas um computador ou um celular smartphone. O objetivo de deixar uma aplicação didática sob os conceitos e funcionamento também estão contemplados. Constatou-se através de testes que o acesso ao dashboard via compartilhamento de link deve ser feito com aqueles que possuam uma conta (gratuita ou não) na plataforma. Para trabalhos futuros deixa-se sugestão para acoplar esta aplicação a um dispositivo real e ainda pode-se aprofundar a pesquisa quanto a segurança dos dados armazenados além da utilização de tecnologia QRCode para maior dinamicidade no acesso da dashboard de monitoramento via smartphone.

A Estação Compacta da FESTO combina 4 circuitos fechados onde estão presentes sensores e atuadores tanto digitais quanto analógicos. Utilizando um CLP é possível usá-los individualmente (controle de nível, controle de pressão, controle de vazão e controle de temperatura) ou em cascata. Esse projeto alcançou seu objetivo de desenvolver o controle em cascata.
